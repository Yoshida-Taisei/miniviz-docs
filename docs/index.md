# Miniviz Docs

## Overview

Miniviz is a library for creating visualizations in the browser. It is built on top of the [D3.js](https://d3js.org/) library and provides a simple API for creating and updating visualizations.

## Getting Started

To get started with Miniviz, you need to install the library. You can do this using npm or yarn.

```bash
npm install miniviz
```
